## How to Run This Project ##
1. Download folder
2. Open in Visual Studio Code
3. Create a new terminal
4. Initialize via npm
```html
  npm init
```
5. Install via npm
```html
  npm install
```
6. Connect to app.js
```html
  node .\app.js
```
7. Type URL in browser
```html
  localhost:8080
```
